What's Inside?

Using an M5Core2 Stack, we programmed an interactive game that required the users to complete simple challenges to be able to "unlock" the box. Challenges included a tapping sequence, angle detector, light sensor, morse code decoder, and sound recognition. Meanwhile, another M5Core would ensure the box wasn't tampered with until all the challenges were complete.

Biofeedback Glove

Working in a group of three students from distinct majors, we used an Arduino, flex sensors, and a mechanical arm to create a glove, allowing a human to control a robot arm by mimicking the user's hand movements. I was in charge of programming the movements in C++ with a simple interface that allowed anyone to wear the glove, calibrate to their hand size and movements, and accurately control the hand.
